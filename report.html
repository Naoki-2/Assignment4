Reproducible Analysis of Cohort Simulated Data

Author

Naoki Hiratsuka

Introduction

This report analyzes cohort simulated data from Assignment 2. The dataset contains individual-level information on smoking status, sex, age, cardiac events, and health care costs. The goal of this report is to summarize the cohort and examine whether smoking, sex, and age are associated with cardiac events and health care costs in the simulated data.

The report also demonstrates a reproducible workflow using Quarto, R, and GitHub. In a reproducible document, the prose, code, tables, and figures are kept together. This makes the analysis easier to update, check, and reproduce.

Methods

Data

The analysis used the cohort simulated data stored in the raw-data folder of this GitHub repository. The dataset contains 5000 observations and 5 original variables.

The original variables are:

Variables in the cohort simulated data
variable_name description
smoke Indicator for smoking status; 1 indicates smoker and 0 indicates non-smoker
female Indicator for sex; 1 indicates female and 0 indicates male
age Age in years
cardiac Indicator for cardiac event; 1 indicates a cardiac event and 0 indicates no cardiac event
cost Simulated health care cost

Notation

Let i = 1, …, n index individuals in the simulated cohort. Let Y_i denote the binary cardiac event outcome for individual i, where Y_i = 1 indicates that a cardiac event occurred and Y_i = 0 indicates that no cardiac event occurred.

Let S_i denote smoking status, where S_i = 1 for smokers and S_i = 0 for non-smokers. Let F_i denote sex, where F_i = 1 indicates female and F_i = 0 indicates male. Let A_i denote age in years. Let C_i denote health care cost.

For the binary cardiac outcome, I estimated the following logistic regression model:

logit[P(Y_i = 1 | S_i, F_i, A_i)] = beta_0 + beta_1 S_i + beta_2 F_i + beta_3 A_i

For health care cost, I estimated the following linear regression model:

C_i = alpha_0 + alpha_1 S_i + alpha_2 F_i + alpha_3 A_i + alpha_4 Y_i + epsilon_i

In these models, the coefficients describe associations in the simulated data. They should not be interpreted as causal effects.

Statistical Analysis

First, I summarized the overall cohort and compared key characteristics by cardiac event status. Second, I created two figures: one showing cardiac event rates by smoking status, and one showing the distribution of health care costs by cardiac event status. Third, I estimated a logistic regression model for cardiac events and a linear regression model for health care costs.

Because these data are simulated, the analysis is intended as an exercise in reproducible statistical reporting rather than evidence about a real population.

Results

Discussion

This report analyzed cohort simulated data using a reproducible Quarto workflow. The analysis found that cardiac events were more common among smokers than non-smokers. In the adjusted logistic regression model, smoking was strongly associated with higher odds of cardiac events. Age was also positively associated with cardiac events, although the association was smaller. Female sex was associated with lower odds of cardiac events in this simulated dataset.

The analysis also found that health care costs were higher among individuals with cardiac events. In the linear regression model, smoking, age, and cardiac event status were positively associated with health care costs, while female sex was negatively associated with costs. These findings are internally consistent with the descriptive results, where individuals with cardiac events had higher smoking prevalence and higher average costs.

The main limitation is that the data are simulated. Therefore, the results should not be interpreted as evidence about a real clinical or policy population. In addition, the models estimate associations rather than causal effects. For example, the coefficient for smoking in the cardiac event model should not be interpreted as the causal effect of smoking on cardiac events. A causal interpretation would require stronger assumptions and a research design that addresses confounding.

The main purpose of this report is to demonstrate a reproducible analytic workflow. The data, code, tables, figures, and prose are contained in a single Quarto document, and the report can be rendered again if the data or code are updated.
